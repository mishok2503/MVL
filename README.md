#### Install

`cd vim_support`  

Add `vimrc` file contents into your `~/.vimrc` file.  

```
mkdir -p ~/.vim/syntax/
cp mvl.vim ~/.vim/syntax/
cp mvl_keywords.txt ~/.vim/

```

Now you can see highlighting and use autocomplete, by pressing `Ctrl+n`, inside our `*.mvl` files.
